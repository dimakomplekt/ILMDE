# Inventor leaf metal dxf export

Макрос iLogic (VBA Inventor) для пакетного экспорта проекций разверток деталей (листовой металл) из текущей сборки. Формирует отдельные .dxf-файлы с указанием масштаба и складывает их в папку, создаваемую под текущую сборку внутри выделенной под экспорт директории.

Заменяет репитативный экспорт n-файлов каждой детали в сборке на нажатие одной кнопки. Работает с любой вложенностью подсборок.

# Состав

1. Рабочий шаблон структуры проекта;
2. Одинаковые по содеранию .bas-файл - для удобства редактирования макроса (к примеру - в VS Code) и .txt-файл - для прямого экспорта в iLogic;
3. .ini-файл настройки экспорта.

В случае нежелания соответствовать моему типовому дереву проекта конструктива у вас имеется прекрасная возможность переписать относительные пути (и не только) в макросе под свои нужды! Перед экспортом (а лучше в процессе моделирования) необходимо проверить "главную сторону" детали из листового металла (кнопочка во вкладке "развертка") - соответствует ли она вашим представлениям о главной стороне (именно её проекция будет выведена в .dxf-файле).

❗❗❗


🚀 Для запуска экспорта необходимо:

1. открыть текущую сборку;
2. импортировать код макроса в модуль iLogic (либо ссылкой на .txt файл, либо копипастом из .bas / .txt файла);
3. запустить модуль из текущей сборки;
4. нажать "OK" в логах вывода деталей (можно убрать, я предпочитаю работать с ними), убедиться, что экспорт реализован успешно.
